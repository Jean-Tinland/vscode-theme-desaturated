## Colors

| Color                                                                   | Hex Code  | Used for                                                                    |
| ----------------------------------------------------------------------- | --------- | --------------------------------------------------------------------------- |
| ![#98A8C5](https://placehold.it/15/98A8C5/000000?text=+) Light blue     | `#98A8C5` | Strings - Comments - Tag brackets                                           |

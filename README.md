# Desaturated Theme

## Introduction

This theme's main goal is to provide a dark, readable & clean experience into your favorite code editor.

There is also 4 variants included :

- No bold
- No italic
- No italic & bold
- Flat (no-borders)

Created by [Jean Tinland](https://www.jeantinland.com)
